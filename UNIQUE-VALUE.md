# Unique Value Proposition

Khyzsh Observability is not a logging library.

Khyzsh Observability is an Event-Centric Observability Toolkit.

Instead of thinking in terms of:

- info
- warn
- error

it thinks in terms of:

- scan
- attendance
- network
- streaming
- security
- audit
- custom events

Every event can automatically generate:

- logs
- metrics
- dashboards
- alerts
- audit trails

from a single source.
